## :page_facing_up: How to use

Clone this repository:

```sh
  $ git clone https://github.com/luc-ribeiro/nlw-04-react-moveit-react.git
```

Install the dependencies:

```sh
  # with npm
  $ npm install

  # with yarn
  $ yarn install
```

Run the command:

```sh
  # with npm
  $ npm start

  # with yarn
  $ yarn start
```

Access the project at `localhost:3000`
